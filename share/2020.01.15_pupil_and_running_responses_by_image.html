Averages for each unique image

The plot below shows the pupil and running responses for each of the 16 images in image set A and B. Some key points for interpretation: * Each line represents the response to a given image, averaged across all ophys sessions (including active and passive sessions) - Vertical bars represent flash times - Blue bars represent times when a single image is present * Gray bars represent times when all stimuli are present, excluding the image represented by the given line * The data used in the average is calculated as (area - area_0)/area_0, where area_0 is the mean area over a +/- * The black line represents the mean response across all images 15 second window surrounding time 0
